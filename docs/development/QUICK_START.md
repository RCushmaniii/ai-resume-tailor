# Quick Start

## Running locally

### 1. Install frontend dependencies

```bash
pnpm install
```

### 2. Start the backend

```bash
pnpm dev:server
```

Backend runs at:

- `http://localhost:5000`

### 3. Start the frontend

```bash
pnpm dev:client
```

Frontend runs at:

- `http://localhost:3000`

### 4. Test the end-to-end flow

- Open `http://localhost:3000/analyze`
- Paste resume + job description
- Click **Analyze Match**
